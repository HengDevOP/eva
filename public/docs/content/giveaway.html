Giveaway System

The giveaway system lets you easily host giveaways in your server. You can set prizes, duration, and winners — all managed by the bot automatically.

Commands

  • !giveaway start <time> <winners> <prize> – Starts a new giveaway in the current channel.
  • !giveaway reroll <messageId> – Rerolls the winners for an existing giveaway.
  • !giveaway end <messageId> – Ends a giveaway early.

Examples

Start a 1 hour giveaway:

!giveaway start 1h 1 Nitro Classic

Reroll a giveaway:

!giveaway reroll 123456789012345678

Notes

  • Time formats supported: s (seconds), m (minutes), h (hours), d (days).
  • Giveaway winners are selected randomly among participants.
  • Only members with the proper permissions can start or end giveaways.
