Moderation

The moderation module helps you keep your community safe and organized. Below are the available commands and their descriptions.

Commands

  • /ban [user] [reason] – Ban a user from the server with an optional reason.
  • /kick [user] – Remove a user from the server.
  • /mute [user] [duration] – Temporarily mute a user for the specified time.
  • /warn [user] [reason] – Issue a warning to a user.

Features

In addition to commands, the moderation system comes with:

  • Automatic logging of all moderation actions
  • Customizable punishments and durations
  • Integration with the warning system
  • Role-based access control for moderators

Tip

Use role permissions to control which moderators can access specific commands.
