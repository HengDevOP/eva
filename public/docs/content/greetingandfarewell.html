Greeting & Farewell System

The Greeting and Farewell system allows you to welcome new members and say goodbye to leaving members automatically. You can customize messages, channels, embeds, and even include media like images or YouTube videos.

Greeting Commands

  • !greeting setchannel <#channel> – Set the channel where greeting messages will be sent.
  • !greeting setmessage <message> – Set a custom greeting message. Use placeholders like {user} and {server}.
  • !greeting embed enable|disable – Enable or disable embed for greeting messages.
  • !greeting embed title <text> – Set the embed title for greeting messages.
  • !greeting embed description <text> – Set the embed description.
  • !greeting embed thumbnail <url> – Set the embed thumbnail (e.g., server icon or user avatar).
  • !greeting embed image <url> – Set an image in the embed. Can be direct image URLs or GIFs.
  • !greeting embed video <YouTube URL> – Embed a YouTube video in the greeting (only works if your bot supports it).

Farewell Commands

  • !farewell setchannel <#channel> – Set the channel where farewell messages will be sent.
  • !farewell setmessage <message> – Set a custom farewell message with placeholders like {user} and {server}.
  • !farewell embed enable|disable – Enable or disable embed for farewell messages.
  • !farewell embed title <text> – Set the embed title for farewell messages.
  • !farewell embed description <text> – Set the embed description.
  • !farewell embed thumbnail <url> – Set the embed thumbnail.
  • !farewell embed image <url> – Set an image in the embed.
  • !farewell embed video <YouTube URL> – Embed a YouTube video in the farewell message.

Tips & Notes

  • You can use placeholders {user} and {server} in messages to dynamically display member and server names.
  • Embeds allow images, GIFs, and even YouTube video links for rich welcome/farewell messages.
  • Ensure the bot has permission to send messages, embed links, attach files, and embed videos in the target channels.
